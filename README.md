# Co robi progaram?

Program ma za zadanie dodać oceny dla danego ucznia, a następnie obliczyć średnią jego ocen. Średnia ulega zmianie przy każdej wprowadzonej ocenie.

# Technologie wykorzystane

* Język Java
* IntelliJ IDEA Community Edition
